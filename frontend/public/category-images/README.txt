Place your category images in this folder.

Naming rule:
- use lowercase slug names
- spaces become "-"
- "&" becomes "and"
- remove special characters

Supported extensions:
- .avif
- .webp
- .jpg
- .jpeg
- .png

Examples:
- Electronics -> electronics.jpg
- Mobiles & Accessories -> mobiles-and-accessories.jpg
- Fruits & Vegetables -> fruits-and-vegetables.jpg
- Beauty & Cosmetics -> beauty-and-cosmetics.jpg
- Tailor & Boutique -> tailor-and-boutique.jpg

Priority order used by the app:
The app uses an internal manifest for exact filenames/extensions.
Current uploaded types can include .avif/.jpg/etc.

If no local image is found, the app falls back to Unsplash automatically.
